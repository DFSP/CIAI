- User
GET /users - Obter todos os utilizadores do sistema
GET /users?search=<criteria> - Obter todos os utilizadores que satisfazem o critério
GET /users/{id} - Obter o utilizador com id {id}
POST /users - Adicionar um utilizador ao sistema. O corpo do Post contém todos os atributos obrigatórios
PUT /users{id} - Atualizar o utilizador com id {id} com os atributos que vão no corpo do PUT
DELETE /users{id} - Remover o utilizador com id {id} do sistema
